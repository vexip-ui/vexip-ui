### Renderer Props

| Name     | Type                      | Description                                                      | Default     | Since |
| -------- | ------------------------- | ---------------------------------------------------------------- | ----------- | ----- |
| renderer | `(...args: any[]) => any` | Set the render function                                          | `null`      | -     |
| data     | `Record<string, any>`     | Set the parameters that need to be passed to the render function | `undefined` | -     |
