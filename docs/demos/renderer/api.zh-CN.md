### Renderer 属性

| 名称     | 类型                      | 说明                       | 默认值      | 始于 |
| -------- | ------------------------- | -------------------------- | ----------- | ---- |
| renderer | `(...args: any[]) => any` | 设置渲染函数               | `null`      | -    |
| data     | `Record<string, any>`     | 设置需要传给渲染函数的参数 | `undefined` | -    |
