### ColorPicker Props

| Name   | Type                  | Description                                                                                                | Default | Since |
| ------ | --------------------- | ---------------------------------------------------------------------------------------------------------- | ------- | ----- |
| props  | `Record<string, any>` | 需要注入的组件属性配置，键值为组件的驼峰命名，值为对应组件的属性配置，注入的属性将覆盖组件原有的属性默认值 | `{}`    | -     |
| locale | `LocaleOptions`       | 需要注入的国际化配置                                                                                       | `{}`    | -     |
