## API

### 属性

| 属性          | 类型    | 说明                                                   | 默认值   |
| ------------- | ------- | ------------------------------------------------------ | -------- |
| vertical      | Boolean | 设置分割线是否为纵向                                   | false    |
| dashed        | Boolean | 设置分割线是否为虚线                                   | false    |
| text-position | String  | 设置嵌入文字的位置，可选值为 `left`、`center`、`right` | 'center' |
| primary       | Boolean | 设置嵌入文字为更显眼的状态                             | false    |
