当你想要在 `<template>` 里使用渲染函数，你可能需要它。

## 代码示例

:::demo renderer/basis

### 简单例子

辅助在模版中使用渲染函数进行内容渲染。

:::

## API

### Renderer 属性

| 名称     | 类型                      | 说明                       | 默认值      | 始于 |
| -------- | ------------------------- | -------------------------- | ----------- | ---- |
| renderer | `(...args: any[]) => any` | 设置渲染函数               | `null`      | -    |
| data     | `Record<string, any>`     | 设置需要传给渲染函数的参数 | `undefined` | -    |
